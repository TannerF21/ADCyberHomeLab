# ADCyberHomeLab
A complete hands-on cybersecurity lab built using Proxmox, Windows Server (AD), Kubuntu Desktop, and Kali Linux.

This project demonstrates:
- Setting up a virtualized enterprise environment  
- Joining Linux to Active Directory  
- Configuring SSH for secure remote access  
- Running offensive security tests from Kali  
- Capturing logs, analyzing attacks, and documenting mitigation  
- Improving defensive posture  
- Writing professional, portfolio-quality documentation  

This lab is designed to be a real-world showcase of:
- System administration  
- Active Directory  
- Linux  
- Networking  
- Offensive security  
- Defensive hardening  
- Documentation skills
